💼 İşləyən BA-lar

İşləyən BA-lar üçün praktik bələdçi

Gündəlik layihə işlərində istinad edə biləcəyin mövzular: requirements, proses modelləşdirmə, stakeholder idarəetməsi, Agile, data və alətlər.

← Öyrənmə mərkəzi

Praktik mövzular

Hər mövzu üçün qısa izah, nə zaman istifadə olunduğu və praktik nümunə.

🎯

Requirements elicitation texnikaları

Tələbləri stakeholderlərdən üzə çıxarmaq üçün metodlar toplusu.

Nə zaman istifadə olunur? Layihənin başında və tələblər dəyişdikcə.

Praktik nümunə: Interview, workshop, observation, document analysis, survey və prototyping — konteksə görə uyğununu seç.

📝

User story və acceptance criteria

Tələbi komandanın anladığı kiçik, dəyərli vahidlərə çevirmək.

Nə zaman istifadə olunur? Backlog hazırlayanda və sprint planlamada.

Praktik nümunə: «Bir müştəri olaraq…» formatı + Given / When / Then qəbul meyarları.

🔀

BPMN proses modelləşdirmə

Biznes proseslərini standart vizual notasiya ilə göstərmək.

Nə zaman istifadə olunur? «As-is» və «to-be» prosesləri təhlil edəndə.

Praktik nümunə: Start event, task, gateway və end event ilə sifariş prosesini çək.

🤝

Stakeholder idarəetməsi

Maraqlı tərəfləri tanımaq, təsnif etmək və gözləntiləri idarə etmək.

Nə zaman istifadə olunur? Layihə boyu, xüsusən başlanğıc mərhələdə.

Praktik nümunə: Power/Interest matrisi və RACI ilə kimin nəyə cavabdeh olduğunu müəyyən et.

🔄

Agile / Scrum BA üçün

Scrum komandasında BA-nın rolu və mərasimlərdə iştirakı.

Nə zaman istifadə olunur? Agile komandalarda gündəlik işdə.

Praktik nümunə: Backlog refinement, sprint planning, review və retrospective-də PO ilə əməkdaşlıq.

📊

Data analitika BA üçün

Qərarları data ilə əsaslandırmaq və hesabat tələblərini formalaşdırmaq.

Nə zaman istifadə olunur? Report/dashboard tələbləri və data keyfiyyəti sualları olanda.

Praktik nümunə: Sadə SQL ilə «son ay sifarişlər»i çıxar, report tələbini dəqiqləşdir.

🛠️

BA alətləri

Gündəlik işdə istifadə olunan əsas alətlər və hər birinin yeri.

Nə zaman istifadə olunur? Sənədləşmə, izləmə və modelləşdirmə zamanı.

Praktik nümunə: Jira (backlog), Confluence (sənəd), Figma (maket), Miro (workshop), Draw.io / Camunda Modeler (BPMN).
